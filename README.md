Laravel 5 package for rapid USSD prototyping and development
## Install

Via Composer

``` bash
$ composer require leyo/rapidussd

##Publish migrations, controllers, models
$ php artisan vendor:publish

##migrate
$ php artisan migrate

##Seed only if you need sample app
$ php artisan db:seed
```
